# Feature Requests
## Login Screen
- Login screen layout needs to be improved and spruced up.
- Remember me checkbox needs to be aligned propery
- The admin username and password shouldn't be referenced as an example in the text boxes
- There should be an option to reset password when a password is forgotten

## Admin
### User Admin
- There should be a way to invite new users and send them a link to join. Users with admin role should be allowed to invite users
- There should be enforced password standards
- There should be a way to use MFA to log in instead of password
### Logs
- We need to add logging to the app to log issues and activities across the app in one place for debugging
- Log retention should be configurable in the settings so logs can be deleted after a certain timeframe
- Logs can be either in a db or in text files, but should be readable within the app itself. Whatever makes sense here

## Movies
- Add a new movies tab to show the movies being tracked (alerts) for each user (the list displayed would be user-specific) with high-level info
- Clicking into a movie should show you more info about the movie and link you to relevent sites

## Alerts
### Alert Detail
- The notification history tile should be a fixed length to not push down the other tiles on the page below it. Because of the number of notifications, the history tile should have page numbers to flip through
